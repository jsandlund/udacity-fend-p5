NoPa Explorer

About

This 'app' is a project for Udacity's Front End Nanodegree Program. Students are asked to use KnockoutJS, the Google Maps API, and at least one additional 3rd party API to translate their neighborhood into an interactive, searchable map. I used APIs from Foursquare & Yelp to pull contextual data about businesses in my neighborhood. Learn more on github.
